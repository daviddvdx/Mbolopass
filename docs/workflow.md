# Workflow

## Patient

1. Crée son compte.
2. Complète son profil.
3. Génère sa carte.
4. Obtient QR / futur NFC.
5. Consulte alertes et résumé IA.

## Urgence

1. QR ou NFC est scanné.
2. Le lien public contient un jeton opaque.
3. Le backend valide le jeton.
4. MboloPass affiche uniquement les informations vitales.
5. L'accès est journalisé.
6. Aucune donnée détaillée n'est exposée.
